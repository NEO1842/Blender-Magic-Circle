# Blender-Magic-Circle

This is a Blender scene for creating a cinematic magical summoning circle effect using procedural particles and glowing energy beams.

A rotating magic circle generates streams of luminous particles while a massive beam of light rises from the center.  
The effect combines bloom, volumetric fog, and animated particle motion to create a mystical fantasy atmosphere.

## -Test Environment-

Blender Version: 3.4  
Rendering Engine: Eevee

## -How to Use-

　1. Download and open Magic-Circle.blend

　2. Play the animation to view the rotating magic-circle effect.

　3. Render the animation as MP4 (2K recommended).

## -Rendering Settings-

Rendering Engine: Eevee

Sampling: 64

Bloom: Enabled  

Volumetrics: Enabled

Resolution: 2160x2160 (2K)  

Output Format: MP4 (H.264)

## -Features-

　* Procedural magical circle animation  
　* Particle-based glowing beam effect  
　* Volumetric fantasy lighting  
　* 2K MP4 rendering support  
　* Lightweight real-time rendering with Eevee

# Blender-Magic-Circle

これは、プロシージャルパーティクルと発光ビームを使用した  
幻想的な魔法陣演出を作成するためのBlenderシーンです。

回転する魔法陣から大量の発光粒子が放出され、  
中央から巨大な光のビームが立ち上がります。

ブルーム、ボリューメトリクス、粒子アニメーションを組み合わせることで、  
神秘的でシネマティックな空間を演出します。

# -テスト環境-

Blenderバージョン：3.4

レンダリングエンジン：Eevee

# -使用方法-

　1. Magic-Circle.blend をダウンロードして開いてください。

　2. アニメーションを再生して、回転する魔法陣演出をご覧ください。

　3. アニメーションをMP4形式でレンダリングしてください（2K推奨）。

# -レンダリング設定-

レンダリングエンジン：Eevee

サンプリング：64

ブルーム：有効  

ボリューメトリクス：有効

解像度：2160x2160 (2K) 

出力フォーマット：MP4（H.264）

# -特徴-

　* プロシージャル魔法陣アニメーション  
　* 粒子ベースの発光ビーム演出  
　* ボリュームライティングによる幻想表現  
　* 2K MP4レンダリング対応  
　* Eeveeによる軽量リアルタイムレンダリング
