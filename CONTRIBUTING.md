# Contributing

Thank you for contributing to Blender-Magic-Circle.

## Ways to Contribute

- Report bugs
- Suggest features
- Improve shaders or effects
- Improve documentation
- Share ideas and optimizations

## Pull Request Process

1. Fork the repository
2. Create a branch
3. Commit your changes
4. Open a Pull Request

## Guidelines

Please keep:
- Clean project organization
- Readable naming
- Optimized Blender files
- Respectful communication

## Notes

Avoid uploading unnecessary large cache files.
